# Petroline API → DOCX Generator

## Опис

Проект для автоматичного отримання транзакцій із Petroline API та генерації `.docx` документів по кожній заправці.

## Ціль

Автоматизувати облік заправок:

* отримувати дані з API
* перевіряти коректність
* створювати документи
* уникати дублювання записів

---

## MVP функціонал

1. Отримання транзакцій через API
2. Обробка списку записів
3. Мапінг даних API → структура документа
4. Перевірка обов’язкових полів
5. Пропуск вже оброблених записів
6. Логування проблемних записів
7. Генерація `.docx` (на наступному етапі)

---

## Структура проекту

```
petroline_docx/
├── main.py              # головний pipeline
├── api_client.py       # робота з API
├── mapper.py           # перетворення API → doc_data
├── validator.py        # перевірка полів
├── storage.py          # processed ids + лог
├── config.py           # конфігурація
├── templates/          # шаблони docx
└── data/               # локальні дані (json)
```
