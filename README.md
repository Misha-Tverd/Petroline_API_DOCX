# Petroline Word Generator

Проект витягує транзакції з Petroline API за задану дату, фільтрує їх по водію і формує заповнену лімітно-забірну картку у форматі `.docx`.

## Налаштування

Скопіюй `.env.example` у `.env` і заповни доступи:

```env
USE_SAMPLE_DATA=false
PETROLINE_LOGIN=
PETROLINE_PASSWORD=
PETROLINE_TOKEN=
DATE_FROM=2026-04-24T00:00:00
DATE_TO=2026-04-24T23:59:59
TARGET_DRIVER=Кондратюк Сергій Анатолійович
FUEL_LIMIT=1000
UNIT_NAME=літр
```

Якщо `PETROLINE_TOKEN` заповнений, програма використовує його як Bearer-токен. Якщо токена немає, програма пробує отримати токен через `PETROLINE_LOGIN` і `PETROLINE_PASSWORD`.

## Запуск

```bash
.venv\Scripts\python.exe -B main.py
```

Готовий документ створюється у:

```text
output/limit_cards/
```

## Поля форми

У документ потрапляють: сільгосппідприємство, структурний підрозділ, одержувач одним рядком `транспорт + ПІБ`, ПІБ одержувача окремим підписом знизу, найменування цінностей, одиниця виміру, ліміт, дата видачі, кількість видано, залишок, разом видано, всього з урахуванням повернутого, підпис одержувача / ПІБ та бухгалтер / ПІБ.
